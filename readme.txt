== Description ==
This plugin is for people who would like to get guest posts.

1. Guest Posts with / without user registration
1. Disable WP-Admin area to registered users
1. Manage post submission editing on the Guest posts page itself
1. Post submission notification
1. Customized Author Profile
1. Check out more information http://kishpress.com/blog/2011/02/25/guest-posting-plugin/

== Installation ==

Installation is very simple, just upload the unzipped files to the plugins directory and activate the plugin. You should go to the settings page and do all the settings first
